## Description

Pokedex Backend with NestJS

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ yarn start:dev:firebase
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
